## 登録方法

###Internet Explorer
ブックマークレットのリンクを右クリックして「お気に入りに追加」を選択する。

###Mozilla Firefox
ブックマークレットのリンクを右クリックして「このリンクをブックマーク」を選択する。

###Google Chrome
ブックマークレットのリンクをブックマークバーへドラッグ & ドロップする。

###スマートフォン
たぶんまともに動かないので、登録はおすすめしません……。
